part5 notes : lets build skill section

Important Note: only functions and classes can be exported using default keyword in react js. Do export default for a datastructure.

make use of card deck of bootstrap

1) create a folder skills in the pages folder
	create a file skills.component.jsx and skills.style.css
	create a file skills-data.js (this file will contain data of our skills)
	its always best to seperate your component logic and data
