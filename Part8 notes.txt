1) For adding particle effect, need to install one library
	npm i react-particles-js

Add particlesOptions.js file in the src folder
	it contains the params for the particle effect
And import above file along with particles effect in the app.js like below

import Particles from 'react-particles-js';
import { particlesOptions } from "./particlesOptions";

below title message component, add the below line for paticle effect

<TitleMessage />
      <Particles className="particles particles-box" params={particlesOptions}  />
